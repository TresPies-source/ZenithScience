## 1. Vision

> v1.3 wires every existing route, component, and backend endpoint into a coherent, usable surface — no new infrastructure, just completion of what's already built.

**The Core Insight:**

The zen-sci-portal has a powerful backend (AgenticGatewayByDojoGenesis) with 60+ API endpoints, 6 MCP servers exposing 17 tools, a memory garden system, MCP app lifecycle, document fetch, tracing, and provider key management — yet the frontend consumes only 9 endpoints. Every route exists (`/workspace/[docId]`, `/app/[appId]`, `/search`, `/settings`, `/chat`, `/memory`, `/tools`, `/pipelines`) but several have stubbed interactions, disconnected data flows, or missing context wiring.

v1.3 is not about adding new backend capabilities. It's about completing the surface: making every button do what the user expects, connecting every component to the data it needs, and exposing the backend power that already exists.

**What Makes This Different:**

This is a **completion release**, not a feature release. Every change in v1.3 connects to an existing endpoint, extends an existing component, or fills a gap visible in the current UI. Zero new Go code. Zero new Rust commands beyond thin wrappers. The work is almost entirely in SvelteKit components, Tauri command wrappers, and API client calls.

The 6 workstreams map exactly to the 6 gaps identified in the v1.2 audit:

1. **Workspace Document View** — section editing, mode transitions, version history
2. **MCP App Companions** — launch/close/list via gateway, iframe communication
3. **Deep Search** — semantic memory search alongside full-text, result click-to-open
4. **Settings Surface** — MCP server status, provider key management, gateway config
5. **Context-Aware Agent** — domain-specialized per document mode, patch intent application
6. **Connected ToolInvoker** — auto-populate from document context, result storage

**Changes to `/app/[appId]/+page.svelte`:**

```svelte
<script lang="ts">
  import { onMount, onDestroy } from 'svelte';
  import { page } from '$app/stores';
  import { launchApp, closeApp, getGatewayUrl } from '$lib/api/tauri.js';
  import { userSession } from '$lib/stores/userSession.js';

  let appId = $derived($page.params.appId);
  let instanceId = $state<string | null>(null);
  let appUrl = $state('');
  let loading = $state(true);
  let error = $state('');

  onMount(async () => {
    try {
      const gatewayUrl = await getGatewayUrl();
      const result = await launchApp(appId!, $userSession?.user_id ?? 'local');
      instanceId = result.instance_id;
      // Resource URL served by gateway
      appUrl = `${gatewayUrl}/v1/gateway/resources?uri=ui://${appId}/preview.html`;
    } catch (err) {
      error = `App not available: ${err}`;
    } finally {
      loading = false;
    }
  });

  onDestroy(async () => {
    if (instanceId) {
      await closeApp(instanceId).catch(() => {});
    }
  });
</script>
```

```typescript
async function executeSearch(query: string, mode: SearchMode) {
  const results: UnifiedResult[] = [];

  if (mode === 'documents' || mode === 'all') {
    const docResults = await search(query);
    results.push(...docResults.results.map(r => ({
      ...r, source: 'document' as const,
      href: `/workspace/${r.document_id}`
    })));
  }

  if (mode === 'memories' || mode === 'all') {
    const memResults = await searchMemory(query, projectId);
    results.push(...memResults.map(r => ({
      id: r.id, title: r.content.slice(0, 80),
      snippet: r.content, source: 'memory' as const,
      relevance_score: r.relevance_score,
      href: `/memory?highlight=${r.id}`
    })));
  }

  // Sort by relevance
  return results.sort((a, b) => (b.relevance_score ?? 0) - (a.relevance_score ?? 0));
}
```

**Apply logic:**
```typescript
async function applyPatchIntent() {
  if (!pendingPatchIntent || !$activeDocument) return;
  const { operation, section_id, content } = pendingPatchIntent;

  if (operation === 'replace' && section_id) {
    await updateSectionContent(section_id, $activeDocument.id, content);
  } else if (operation === 'append') {
    const lastSection = $activeDocument.sections.at(-1);
    if (lastSection) {
      const newContent = lastSection.content + '\n\n' + content;
      await updateSectionContent(lastSection.id, $activeDocument.id, newContent);
    }
  } else if (operation === 'insert' && section_id) {
    await updateSectionContent(section_id, $activeDocument.id, content);
  }

  // Refresh document
  const updated = await getDocument($activeDocument.id);
  activeDocument.set(updated);
  pendingPatchIntent = null;
}
```
